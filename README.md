# 땅굴파기 게임 🎮

Python과 Pygame으로 만든 현대적이고 세련된 땅굴파기 어드벤처 게임입니다.

## ✨ 디자인 특징

- **현대적인 UI/UX**: Tailwind CSS 스타일의 색상 팔레트 적용
- **카드 기반 디자인**: 둥근 모서리와 그림자 효과
- **부드러운 애니메이션**: 글로우, 펄스 효과
- **그라디언트 배경**: 깊이감 있는 비주얼
- **세련된 타이포그래피**: 계층적 정보 구조

## 게임 소개

용사가 되어 지하 30층까지 땅굴을 파내려가는 게임입니다!
각 층마다 몬스터들이 배치되어 있으며, 아래로 내려갈수록 더욱 위험해집니다.

## 게임 규칙

- 🌍 **시작 위치**: 몬스터가 없는 안전한 지상에서 시작합니다
- 🏔️ **총 51개 층**: 지상 1층 + 지하 1층부터 50층까지
- ⛏️ **땅굴 파기**: L 키를 눌러 현재 위치에 구멍을 파세요 (1초 소요)
- 🕳️ **구멍을 통한 이동**: 파인 구멍 위에서만 S 키로 아래층으로 이동할 수 있습니다

### 👾 **몬스터** (5종류)
  - 지하 1~10층: 해골 병사 (흰색)
  - 지하 11~20층: 박쥐 (보라색)
  - 지하 21~30층: 좀비 (녹색)
  - 지하 31~40층: 드라큘라 (빨간색) ⭐ NEW!
  - 지하 41~50층: 오크 (진한 녹색) ⭐ NEW!
  - ⚡ 아래층으로 갈수록 속도 증가
  - 🔀 41층부터는 랜덤 방향 전환!

### 🎁 **기믹 시스템** (5종류)
1. **노란색 - 순간이동** 🌟
   - 4층 아래로 순간이동
   - 위치: 20층, 28층, 42층
   
2. **주황색 - 투명화** 👻
   - 2층 동안 무적 상태
   - 위치: 5층, 13층, 34층, 45층
   
3. **보라색 - 감속** 🐌
   - 3초간 이동속도 50% 감소
   - 위치: 5층, 14층, 31층, 46층
   
4. **분홍색 - 가속** ⚡
   - 3초간 이동속도 50% 증가
   - 위치: 8층, 24층, 37층
   
5. **검정색 - 마비** 😵
   - 1초간 움직임 정지
   - 위치: 3층, 11층, 22층, 31층, 45층

- 💀 **게임 오버**: 몬스터와 부딪히면 게임이 종료됩니다
- 👸 **클리어**: 지하 50층에서 공주를 구출하면 클리어! 💖
- ⏱️ **타이머 시스템**: 게임 시작부터 클리어까지의 시간이 측정됩니다
- 🏅 **랭킹 시스템**: 상위 1~3위 기록은 명예의 전당에 이름을 새길 수 있습니다

## 조작법

| 키 | 기능 |
|---|---|
| `A` | 왼쪽으로 이동 |
| `D` | 오른쪽으로 이동 |
| `S` | 아래층으로 이동 (땅굴이 파진 상태에서만) |
| `Space` | 점프 (위층으로 1층 올라가기) |
| `L` | 땅굴 파기 (약 1초 소요) |
| `V` 또는 `VIEW 버튼` | View 모드 ON/OFF (전체 맵 보기) ⭐ NEW! |
| `↑` `↓` (View 모드) | 카메라 위아래 이동 |
| `마우스 휠` (View 모드) | 카메라 스크롤 |
| `R` | 게임 재시작 (게임 오버/클리어 시) |
| `ESC` | 게임 종료 |

## 설치 및 실행

### 1. 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 게임 실행

**방법 1: 배치 파일 사용 (가장 쉬운 방법)**
- `게임실행.bat` 파일을 더블클릭

**방법 2: PowerShell 스크립트 사용**
- `게임실행.ps1` 파일을 우클릭 → "PowerShell로 실행"

**방법 3: 명령어로 실행**
```bash
python tunneling_game.py
```

## 🌐 사내 공유용: 웹(브라우저)에서 바로 실행하기 (EXE 없이)

보안 정책상 EXE 공유가 어렵다면, **웹 버전**으로 공유할 수 있습니다.

### 1) 웹 빌드 생성

- `web_build.bat` 더블클릭 (권장, 파일명 깨짐 방지)
- (대안) `웹빌드실행.bat` 더블클릭
- 산출물: `build\web\index.html`, `build\web\tunnelinggame_app.apk`

### 2) 로컬에서 바로 플레이(다운로드 배포)

- (권장) **Python 없이 실행**: `web_run_local_ps.bat` 실행 → 브라우저에서 `http://localhost:8000` 접속
- Python이 설치되어 있다면: `web_run_local.bat` 실행
- (대안) `웹로컬실행.bat` 실행

### 3) 사내 웹서버에 올려 URL로 공유(추천)

- `build\web` 폴더를 사내 정적 호스팅에 업로드 후 `index.html` 링크 공유

## 🌍 GitHub Pages / Vercel로 링크 배포하기 (추천)

로컬 실행이 사내 보안/네트워크 정책에 막히는 경우, **공개 URL**로 배포하면 훨씬 안정적으로 동작합니다.

### A) GitHub Pages (가장 간단 + 자동 배포)

1. 이 프로젝트를 GitHub 저장소로 푸시
2. GitHub → Settings → Pages
3. **Build and deployment**를 “GitHub Actions”로 설정
4. `main` 브랜치에 푸시하면 워크플로우(`.github/workflows/deploy_web.yml`)가 자동으로 웹 빌드 후 배포

### B) Vercel (정적 호스팅)

1. Vercel에 GitHub 저장소 연결
2. 프로젝트 설정에서 그대로 Deploy
   - `vercel.json`이 **빌드 없이** `build/web`을 정적 결과물로 서빙하도록 설정돼 있습니다.
3. 배포 전에 로컬에서 `web_build.bat`를 실행해 **최신 `build/web`이 생성된 상태**로 푸시하는 것을 권장합니다.

### 개발용 테스트 기능(T 키) 정책

- 배포본(웹/EXE)에서는 **기본적으로 동작하지 않음**
- 개발자가 로컬에서만 활성화:
  - PowerShell: `$env:TUNNELINGGAME_DEVTOOLS="1"`
  - CMD: `set TUNNELINGGAME_DEVTOOLS=1`

4. 🕳️ **구멍 위치 확인**: 파인 구멍 위에 있어야만 아래층으로 이동할 수 있습니다

6. 📍 **전략적으로 파기**: 각 층에서 구멍을 파는 위치를 신중하게 선택하세요
7. 🎁 **기믹 활용**: 빛나는 기믹을 잘 활용하면 게임이 훨씬 쉬워집니다
   - 노란색: 4층 점프로 빠른 진행
   - 주황색: 투명화로 안전 통과
   - 분홍색: 가속으로 몬스터 회피
   - ⚠️ 보라색/검정색: 피하세요!
8. 👁️ **층 번호 확인**: 화면 왼쪽에 현재 층 번호가 표시됩니다
9. 🏃 **속도 증가 주의**: 아래층으로 갈수록 몬스터가 빨라지니 신중하게 움직이세요
10. 🔀 **41층 이상**: 몬스터가 갑자기 방향을 바꿀 수 있으니 더욱 조심하세요!
11. ⏱️ **빠른 클리어**: 화면 중앙의 타이머를 보며 최단 시간 기록에 도전하세요!
12. 👸 **공주 구출**: 지하 50층에 도착하면 공주를 구출합니다!
13. 🏆 **명예의 전당**: 1~3위 안에 들면 이름을 새길 수 있습니다

## 랭킹 시스템

게임을 클리어하면:
1. 화면 중앙에 클리어 타임이 표시됩니다
2. 상위 1~3위 안에 들면 **이름 입력 화면**이 나타납니다
3. 이름을 입력하고 Enter 키를 누르면 명예의 전당에 등록됩니다
4. 랭킹은 `ranking.json` 파일에 자동 저장됩니다

### 랭킹 화면
- 🥇 1위: 금메달
- 🥈 2위: 은메달
- 🥉 3위: 동메달

## 🔨 게임 빌드하기 (개발자용)

### 실행 파일(.exe) 만들기

```bash
# 1. PyInstaller 설치
pip install pyinstaller

# 2. 빌드 실행
pyinstaller --clean --onefile --noconsole --name "TunnelingGame" tunneling_game.py

# 3. 결과 확인
dist/TunnelingGame.exe  ← 생성된 실행 파일
```

또는 제공된 배치 파일 사용:
```bash
빌드실행.bat
```

자세한 내용은 `배포가이드.md` 참조
